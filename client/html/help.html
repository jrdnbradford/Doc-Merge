Sheet & Template Doc Setup

Sheet

Copy the Google Sheet and add in your data. Ensure that the header is set in the first row.

Template Doc

Your template Doc should have fields corresponding to your Google Sheet's headers, surrounded by double less-than and greater-than signs (for example, <<First Name>> or <<Zip Code>>). These parts of your Doc will be replaced with the corresponding data in the sheet. The data in the header and rows of your Google Sheet should not have less-than and greater-than signs.

Configuration Fields

Google Doc Template URL

The URL to the Google Doc Template. You must have at least view access to the Doc.

Merge Folder Name

The name of the folder that will be created in the root of your Google Drive to hold your merged Docs. By default it will be set as the name of the template's title plus "Merge".

Merge Type

The format of the files to be merged. By default it is set to merge as PDFs.

File Name

The sheet column by which to name the merged files. The available options are the names of the columns in the sheet. If the sheet has a Last Name column, this will be set as the default. If you keep this default setting, each merged file will be named after the data in the Last Name column.

Share Via Drive

This option is only available if your Google Sheet has an email column that can be used to grant view access to the resulting documents. If you choose to merge as PDFs, an email notification is sent notifying the recipient that access has been granted to the PDF. If you choose to merge as Docs, no email notification is sent (due to a limitation with Google's API), but view access is still given to the account.

Quotas & Limitations

Google has quotas and limitations regarding how many documents you can create daily and how long scripts (like the one in this Google Sheet) can run.
